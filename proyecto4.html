Implementación de una Infraestructura de Red Segura en una Pequeña Empresa

El proyecto consiste en el diseño, configuración e implementación de una red segura para una pequeña empresa ficticia, con un enfoque en la seguridad de datos, la eficiencia operativa y la protección contra amenazas externas. La infraestructura incluye servidores, estaciones de trabajo, medidas de seguridad y automatización básica, diseñada para escalar con la empresa a medida que crecen sus necesidades y sus recursos.

Objetivos del proyecto

  • Diseñar una red local (LAN) con segmentación mediante VLANs.
  • Implementar un servidor de directorio activo (Active Directory) para la gestión de usuarios y permisos.
  • Configurar un servidor de archivos compartidos con control de acceso basado en roles (RBAC).
  • Desplegar un sistema de copias de seguridad automatizadas para garantizar la recuperación de datos.
  • Configurar un firewall para proteger la red contra ataques externos.
  • Monitorizar la red con herramientas de análisis y detección de intrusos (IDS).

Tecnologías Usadas:

  • Servidor de Directorio Activo (Active Directory): Microsoft Windows Server
  • Servidores de Archivos Compartidos y Copias de Seguridad: Windows Server, Samba (Linux)
  • Firewall: Firewall de hardware gestionado (e.g. Palo Alto, Fortinet), o soluciones de software como PFsense.
  • Herramientas de Monitoreo y Análisis de Red: Nagios, Zabbix, Elastic Stack (Elasticsearch, Logstash, Kibana)

Proceso de Implementación

  • Diseño de la Red LAN: 
    • Crear diagramas de red que muestren la estructura de las VLANs, la topología de red y los equipos clave (servidores, switches, routers).
    • Configurar un conmutador gestionado para implementar VLANs con políticas de seguridad.

  • Configuración de Active Directory: 
    • Instalar y configurar un servidor de Windows Server para actuar como controlador de dominio.
    • Establecer políticas de seguridad (contraseñas, bloqueos de cuenta, configuración de sesión) mediante GPO (Group Policy Object).
    • Automatizar la creación de cuentas de usuario y grupos mediante scripts de PowerShell.

  • Configuración de Servidores de Archivos Compartidos y Políticas de Acceso Basado en Roles (RBAC): 
    • Configurar un servidor de archivos compartidos en Windows Server.
    • Crear grupos de seguridad y asignar permisos a carpetas específicas basadas en roles y usuarios.
    • Implementar políticas de copia de seguridad automatizadas para recursos compartidos.

  • Configuración del Firewall: 
    • Configurar reglas de seguridad en un firewall de hardware o software para filtrar el tráfico interno y externo.
    • Crear políticas de seguridad básicas para permitir el acceso a Internet solo a través de servicios específicos (e.g. HTTP, HTTPS).

  • Implementación de Herramientas de Monitoreo y Análisis: 
    • Configurar Nagios o Zabbix para monitorear el rendimiento de la red, el estado de los servidores y los recursos compartidos.
    • Configurar alertas automáticas para detectar intrusiones, fallos de red y actividades sospechosas.

  • Desplegar Copias de Seguridad Automatizadas: 
    • Programar tareas de copia de seguridad diarias que realicen backups completos e incrementales.
    • Almacenar copias de seguridad en un dispositivo externo seguro o en la nube para facilitar la recuperación de datos en caso de desastre.
